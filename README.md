# auto-dhcp-server
Проект создан для упрощения настройки DHCP-сервера начинающим администраторам. Он подойдёт для учебных задач, лабораторных работ, виртуальных машин и тестовых стендов.

Цель проекта:
Проект создан для упрощения настройки DHCP-сервера начинающим администраторам.
Он подойдёт для учебных задач, лабораторных работ, виртуальных машин и тестовых стендов.


Возможности:
- Автоматическая установка пакета isc-dhcp-server\
- Проверка запуска от root\
- Запрос сетевого интерфейса у пользователя
- Валидация существования интерфейса
- Автоматическое создание конфигурации:
- Для интерфейса enp0s3 создаётся подсеть 10.0.2.0/24
- Для всех остальных интерфейсов — подсеть 192.168.100.0/24
- Настройка файла /etc/default/isc-dhcp-server
- Перезапуск службы DHCP

Возможность:
- Просмотреть статус сервера
- Проверить, слушается ли порт 67 (DHCP)

Требования:
- Debian/Ubuntu или любой дистрибутив с apt
- Права суперпользователя (root)
- Пакет isc-dhcp-server

Пример работы:
- Скриншоты находятся в папке images
